# Lucas Forte Personal

Landing page em Astro para portfólio, criada para um personal trainer fictício com foco em conversão via WhatsApp e formulário.

## Stack

- AstroJS
- SCSS
- React
- `lucide-react`
- `framer-motion`
- `gsap`

## Rodando localmente

```bash
npm install
npm run dev
```

## O que já está pronto

- Hero com CTA principal
- Seção de dores, solução, serviços, depoimentos e sobre
- CTA final com WhatsApp e formulário
- Visual dark com acento laranja vibrante
- Animações com GSAP e Framer Motion
